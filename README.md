# 🌐 Portfolio Website

This is my personal portfolio website showcasing my projects, skills, certifications, and contact information.

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/balkrushnanaik/Personal_Portfolio
```

### 2. Navigate to the Project Folder

```bash
cd Personal_Portfolio
```

### 3. Go to the Frontend Directory

```bash
cd frontend
```

### 4. Install Dependencies

```bash
npm install
```

### 5. Start the Development Server

```bash
npm run dev
```

### 6. Open in Browser

After the server starts, open the URL displayed in the terminal (usually):

```
http://localhost:5173
```
